# 김비서 - 통합 비즈니스 대시보드 시스템

> **프로젝트명**: 김비서  
> **설명**: 매출, 프로젝트, 일정, 회의 등 전사 비즈니스 데이터를 한곳에서 관리하는 통합 대시보드 플랫폼  
> **버전**: 1.0.0  
> **제작일**: 2026-04-09

---

## 🎯 주요 기능

### 1. **통합 대시보드**
- 📊 실시간 매출 현황 및 통계
- 📋 우선순위별 업무 목록 관리
- 📅 주간 일정 조회
- 📈 진행 중인 프로젝트 진행률 표시
- 🌓 라이트/다크 모드 지원

### 2. **데이터 분석**
- 📈 월별 매출 추이 차트
- 🏆 제품별 판매량 분석
- 🌍 지역별 매출 현황
- 📊 카테고리별 매출 비율

### 3. **회의 및 보고**
- 💬 회의 기록 및 액션 아이템 관리
- 📝 웹사이트 분석 리포트
- 📊 스프레드시트 데이터 분석
- 📋 계약 이후 프로세스 체크리스트

### 4. **프로세스 관리**
- 🔄 5단계 업무 프로세스: 기획 → 제작 → 검토 → 배포 → 분석
- ✅ 기관별 계약 이후 프로세스 (18개 체크리스트)

## 📁 파일 구조

```
김비서/
├── CLAUDE.md                          # 프로젝트 설명서
├── dashboard.html                     # 📍 메인 대시보드 페이지
├── chart.html                         # 📊 매출 차트 분석
├── meeting-result.html                # 💬 회의 기록 및 액션 아이템
├── process.html                       # 🔄 업무 프로세스 다이어그램
├── diagram.svg                        # 📋 SVG 프로세스 다이어그램
├── report.html                        # 📝 웹사이트 분석 리포트
├── spreadsheet-report.html            # 📊 스프레드시트 분석 리포트
├── 기관별프로세스.html                 # ✅ 계약 이후 프로세스
│
├── 김비서-데이터/                      # 📂 데이터 폴더
│   ├── 매출데이터.csv                  # 제품별, 지역별 매출 데이터 (30건)
│   ├── 업무목록.csv                    # 진행 중인 업무 및 일정
│   ├── 주간일정.txt                    # 이번 주 일정 및 회의
│   ├── 프로젝트현황.csv                # 진행 중인 프로젝트 진행률
│   └── 회의록.txt                      # 최신 회의 내용
│
└── 정리해줘/                           # 📂 문서 및 자료
    ├── 보고서/                         # 보고서 초안, 최종본
    ├── 메모/                           # 기획 메모, 아이디어 메모
    ├── 업무/                           # 할일, 영수증 정리
    └── 기타/                           # 링크 모음, 임시 파일
```

## 🛠 사용 기술

### **프론트엔드**
- **HTML5** - 시맨틱 마크업
- **CSS3** - Glassmorphism 디자인 패턴
  - `backdrop-filter: blur(10px)` 효과
  - CSS Grid & Flexbox 레이아웃
  - CSS 변수를 이용한 다크모드 지원
  - 반응형 웹디자인

### **JavaScript**
- Vanilla JS (라이브러리 미사용)
- **localStorage** - 사용자 테마 설정 저장
- **Canvas API** - 차트 시각화
- **CSV 파싱** - 데이터 처리
- **DOM 조작** - 동적 UI 업데이트

### **데이터**
- **CSV 파일** - 매출, 업무, 프로젝트 데이터
- **TXT 파일** - 회의록, 일정 정보
- **SVG** - 벡터 그래픽 다이어그램

### **DevOps**
- **Git** - 버전 관리
- **GitHub** - 저장소 호스팅

## 🚀 사용 방법

### 1. 로컬 실행
```bash
# 파일을 브라우저에서 열기
open dashboard.html

# 또는 간단한 웹 서버 실행
python -m http.server 8000
# http://localhost:8000 접속
```

### 2. 네비게이션
- **대시보드**: 전체 현황 요약
- **매출 차트**: 상세 판매 분석
- **회의 기록**: 액션 아이템 확인
- **업무 프로세스**: 프로세스 플로우
- **분석 리포트**: 웹사이트 분석
- **스프레드시트 분석**: 매출 데이터 분석

### 3. 테마 전환
우측 상단의 🌓 버튼으로 라이트/다크 모드 전환 (localStorage에 저장)

## 🎨 디자인 특징

### **Glassmorphism**
```css
backdrop-filter: blur(10px);
background: rgba(255, 255, 255, 0.1);
border: 1px solid rgba(255, 255, 255, 0.2);
```

### **색상 스킴**
- **라이트 모드**: #ffffff 배경, #1a1a1a 텍스트
- **다크 모드**: #0f0f23 배경, #e0e0e0 텍스트
- **주요 색상**: #667eea (보라색), #764ba2 (자주색)

### **반응형 디자인**
- 모바일, 태블릿, 데스크톱 최적화
- CSS Grid와 Flexbox로 유연한 레이아웃
